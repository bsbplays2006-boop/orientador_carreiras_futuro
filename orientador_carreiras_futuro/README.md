# Sistema Inteligente de Orientação de Carreiras

Este projeto é uma ferramenta de orientação profissional voltada para carreiras do futuro, desenvolvida em Python com programação orientada a objetos.  
O sistema analisa perfis, registra competências técnicas e comportamentais, e gera recomendações de carreiras, trilhas de aprendizado e áreas de aprimoramento.

## Descrição e Propósito

O objetivo é conectar lógica de programação, automação e desenvolvimento humano, preparando pessoas para o trabalho do futuro.  
O sistema permite:

- Organizar e analisar perfis profissionais com base em competências.
- Identificar lacunas e sugerir áreas de aprimoramento.
- Recomendar carreiras e trilhas de aprendizado personalizadas.
- Utilizar listas, tuplas, dicionários e classes para modelagem e análise de dados.
- Interagir por meio de interface textual (CLI) simples.
